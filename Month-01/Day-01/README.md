## How to Run

**Python:**
```bash
python Python/word_counter.py
python Python/student_management.py
python Python/bank_account.py
```

**DSA (Java):**
```bash
javac DSA/two_sum.java && java -cp DSA two_sum
javac DSA/contains_duplicate.java && java -cp DSA contains_duplicate
javac DSA/valid_anagram.java && java -cp DSA valid_anagram
```

**AI Chatbot:**
```bash
cd AI
pip install -r requirements.txt
# Create .env with: GROQ_API_KEY=your_key_here
python chatbot.py
```
